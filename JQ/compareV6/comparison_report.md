## 4. 差异根因分析

### 4.1 P0：JQ 导出数据可能异常

**证据**:
- `transaction.csv` 仅 7 条记录（全为买入，仅 1 只不同股票）
- `result_1 (3).csv` 中 `当日买请` 列仅 3 天有非零值，总和 301,002
- 这意味着 JQ 平台记录的 **总成交额仅30万元**，在 300,000 初始资金下约等于 1 倍换手率
- 本地回测的买入成交额日均 5-15 万，4 年累计远超百万

**可能原因**: 
- JQ export 功能对某些列（buy/sell volume）统计口径不同
- 文件名 `result_1 (3).csv` 暗示这可能是多次运行中的某一次，可能不是最终版本
- 或者策略在 JQ 上触发了某种限制（如 `set_universe` 失败、`history()` 返回空数据等）

### 4.2 P0：前复权数据差异（用户已指出）

两个平台使用不同的数据供应商，前复权算法不同：

| 环节 | 影响 |
|------|------|
| **5 日跌幅计算** | 不同复权 → 不同 `close/close.shift(5)-1` → 完全不同的 MR 信号触发 |
| **RSI(6) 计算** | 不同复权 → 不同 `(gain_avg/loss_avg)` → 不同 RSI 值 |
| **入场价格** | 复权后价格差异 → 成本基准不同 |
| **-2% 止损触发** | 不同复权 → 止损触发时机不同 |

对于均值回归策略（核心逻辑是"跌多了买"），复权差异会导致选出**完全不同的股票**。JQ 数据中触发了 MR 信号的股票，在金玥数据中可能未触发，反之亦然。

### 4.3 P1：股票池构成差异

| 维度 | JQ 脚本 | 本地 |
|------|---------|------|
| 检测池 | ALL = **HS300 + CYB_STAR_50** (~352只) | ALL = **HS300 + CSI1000 + CYB_STAR_50** (~1339只) |
| 交易池 | 同上 (~352只) | HS300 + CYB_STAR_50 (~352只) |
| 成分股更新 | JQ 实时按日期更新成分股 | 静态列表（2026-06 snapshot） |

本地检测池更大 → 有更多候选标的 → 更容易找到超卖反弹机会。JQ 仅 352 只检测可能错过大量信号。

RSI<20 阈值下，微小差异就可能导致信号有无的变化。

### 4.5 P2：状态识别差异

| 环节 | JQ 脚本 | 本地 |
|------|---------|------|
| 滞后方式 | `idx = -4` (固定 T-3) | `shift(3, freq='B')` |
| 数据源 | JQ 沪深300实时数据 | 金玥 hs300 预计算 |
| 指标依赖 | MA20/MA60 用 JQ rolling | 预计算的 MA20/MA60 |

两种滞后方式在月末/月初边界可能产生 1-2 天的差异，进而影响 CRASH → BEAR → SIDEWAYS 等状态切换，进一步影响仓位乘数。

## 5. 结论与建议

### 5.1 核心结论

1. **JQ 与本地回测结果存在数量级差异**（10.07% vs 305.57%），无法用正常跨平台误差解释
2. **JQ 的 transaction.csv 仅 7 条记录**，强烈暗示该结果文件可能对应一次异常/不完整运行
3. **前复权数据差异 + 股票池大小差异** 是两个确定性的贡献因素，但单独每个都不足以解释 295pp 的差距
4. **本地回测结果与策略 spec 一致**（spec 声称 39.2% 年化 / -13.9% DD，本地 41.05% / -14.16%）

### 5.2 建议下一步

| 优先级 | 行动 | 预期效果 |
|--------|------|---------|
| **P0** | 确认 JQ 结果文件是否为正确运行的输出 | 如果是异常运行，重新在 JQ 上跑一次 |
| **P0** | 将本地检测池缩小到与 JQ 完全一致（仅 HS300+CYB_STAR_50，不含 CSI1000）单独跑一次 | 量化股票池差异的影响 |
| **P1** | 在本地对几只信号触发差异的代表性股票做逐日对比 | 量化前复权差异的影响 |
| **P1** | 统一 RSI 计算（本地用 Wilder smoothing 重算 RSI6） | 消除 RSI 实现差异 |
| **P2** | 本地输出每日信号详情（哪些股票触发了哪种信号），供与 JQ 日志对比 | 精确定位信号层面的差异 |
