1. 策略概览
ATOS（Adaptive Trend-Oscillation Strategy）是一套纯技术面量化做多系统，核心是通过4种市场状态识别动态切换参数、仓位与买卖逻辑，在牛市中重仓趋势跟踪、震荡市中波段与均值回归、熊市中轻仓反弹、暴跌中强制空仓，同时专门设置震荡下行子模式以消除缓慢阴跌环境下的反复止损。策略年化目标≥15%，最大回撤控制在15%以内，波动率≤18%，夏普≥1.2。全策略采用日线数据，不使用机器学习、基本面及另类数据，仅依赖价格、成交量与少数技术指标，参数体系经过分层与自适应设计，具备较强的泛化能力与防过拟合特性。

2. 市场环境识别
2.1 四状态定义
基于沪深300指数（代码：000300）的均线、波动率、价格幅度，将市场划分为：

状态	判定条件（必须同时满足）	目标仓位
BULL	① 收盘价 > MA60
② MA20 > MA60
③ 近20日涨幅 ≥ 5%
④ 20日年化波动率 ≤ 30%	80%-100%
SIDEWAYS	① 价格在 MA60 ± 5% 区间内运行（过去20日最高最低点均未突破该区间）
② MA20 与 MA60 的绝对值差距 < 2%
③ 近20日涨幅在 -5% ~ 5%	50%-70%
BEAR	① 收盘价 < MA60
② MA20 < MA60
③ 近20日涨幅 ≤ -5%
④ 不属于 CRASH	10%-20%
CRASH	① 收盘价 < MA60
② 近5日跌幅 ≥ 8% 或 近20日跌幅 ≥ 15%
③ 20日波动率突然放大至 > 35% 且 近5日跌幅 > 5% (可选快速判定)	0%-10%（强制清仓）
2.2 状态确认与冷却机制
常规切换：新状态条件需连续3个交易日满足，才正式切换。

急速切换至CRASH：上述CRASH条件一旦满足，无需等待，次日开盘即切换，且触发强制清仓。

冷却期：状态切换后，至少维持5个交易日不变，防止频繁抖动。

震荡下行叠加：在 BEAR 或 SIDEWAYS 状态下，若同时满足震荡下行四维检测（见第6节），则启用震荡下行专项规则，覆盖原状态参数，但状态本身不做变更。

2.3 每日输出
每日收盘后计算并记录：当前状态、确认倒计时、目标仓位上限。

4. 多层止损体系
卖出优先级：CRASH强制清仓 → 移动止盈 → 分批止盈 → 时间止损 → 硬止损。

4.1 CRASH 强制清仓
状态确认转为CRASH后，次日开盘立即清空所有持仓，并禁止新开仓，直至退出CRASH。

4.2 移动止盈（核心保护）
持仓期间记录每日最高收盘价 H_entry_high。

止盈价 = H_entry_high - k_trail × ATR(14)，只允许向上移动。

一旦收盘价 ≤ 止盈价，次日开盘卖出。

k_trail 根据状态自适用（见3节）。

4.3 分批止盈
达到浮盈阈值后，按规则减仓，剩余仓位继续按移动止盈管理。

4.4 时间止损
持仓天数超过状态规定最大周期，且浮动盈亏 ≤ 0，则收盘价卖出。

4.5 硬止损（仅BULL状态）
亏损达到 -8%（相对于买入成本），无条件卖出。

其他状态下不使用固定比例硬止损，因为震荡和熊市中极易反复触发。

5. 震荡下行专项
5.1 四维检测（全部条件同时满足）
60日累计跌幅：-5% ～ -20%（不含崩盘）。

低波动率：60日年化波动率 < 18%。

MA60走平：MA60 的 20日斜率绝对值 < 0.5%/月（即约0.02%/日）。

均线空排：MA20 < MA60。

5.2 专项规则（覆盖原状态参数）
仓位：总仓位 ≤ 20%，单只 ≤ 3%。

持仓周期：≤ 5 个交易日。

开仓条件：当日涨幅 ≥ 2% 且 量比 ≥ 1.5 倍，同时该股前5日曾创20日新低（反弹捕捉）。

止盈：浮盈 3% 卖出 1/3，浮盈 5% 全部清仓。

禁止项：禁用硬止损、突破追高买入、均值回归网格、加仓。

退出震荡下行模式：从进入该模式起计算累计收益，当累计收益 > 0 且 某日出现放量大阳线（涨幅>2%，量比>1.5），自动退出，恢复原状态规则；或持续8周（40个交易日）仍未退出，则强行退出并重新评估市场状态。

6. 五层可调参数结构
层级	说明	示例参数
L1 核心不可调	策略根基，不可修改	最大回撤上限15%，CRASH空仓机制，风控阶梯
L2 核心可调	适度调整优化整体风格	基础止损/止盈倍数、仓位基准、选股因子范围
L3 状态特化	不同市场状态的不同参数集	见第3节分状态参数表
L4 风格特化	针对市值/板块风格（此处简化）	预留，暂未使用（纯量价）
L5 因子特化	单个因子或信号的微调	具体因子权重、金叉阈值微调
五大自适应参数（跨层级联动）：

止损倍数 k：k = base_k × (target_vol / max(realized_vol, 0.08))，限制在 [1.2, 4.0]。

持仓周期：根据滚动20日胜率，胜率>60%则延长2天，胜率<45%则缩短2天。

仓位上限：乘以市场宽度系数 breadth = (沪深300成份股中站上MA60的比例)，范围0.5~1.0。

开仓阈值：因子得分阈值 = base_threshold + (0.5 - rolling_winrate_20d) × 0.3。

因子权重：BULL增加动量权重，SIDEWAYS增加反转权重，BEAR增加低波/质量权重。

7. 选股模型
7.1 因子池（4类12因子）
趋势类：

trend_ma60：收盘价 / MA60 - 1

ma_slope_60：MA60的20日线性回归斜率（年化）

adx_14：平均趋向指数（14日）

price_above_ma120：价格 > MA120（0/1）

动量类：

ret_20d：过去20个交易日收益率

ret_60d：过去60日收益率

sharpe_60d：60日日收益率年化夏普

量价类：

vol_ratio_5_20：5日均量 / 20日均量

turnover_stability：20日换手率标准差（负向）

corr_ret_vol：最近20日收益率与成交量的相关系数

形态类：

rsi_14：14日RSI（经映射：远离极值）

bb_width：布林带(20,2)宽度 / MA20

near_high_60：（60日最高 - 收盘）/ 60日最高

7.2 因子处理
去极值：MAD法，阈值 5。

标准化：Z-score（滚动3年窗口）。

中性化：对市值（取对数）回归取残差（用沪深300成分股总市值）。

7.3 合成与动态权重
合成方式：IC加权（基于滚动120日IC），每月更新权重。

动态调整：牛市中动量因子权重×1.5，震荡市中量价反转因子权重×1.5，熊市中低波动/防御因子权重×1.5。

8. 买卖信号
8.1 买入信号（至少6类，需满足状态对应的开仓阈值）
均线金叉：MA5上穿MA10，且成交量放大 ≥ 1.2倍20日均量。确认：次日收盘仍站稳MA10。

KDJ金叉：K上穿D，且K < 80，D < 70。确认：次日K值继续上升。

布林带下轨反弹：昨日收盘价 ≤ 布林下轨，今日收阳线且成交量 > 1.2倍均量。立即确认。

Donchian突破：收盘价突破20日最高价，且ADX(14) > 20。确认：次日未跌回原区间。

MACD底背离：价格创20日新低，MACD柱未创新低，随后MACD金叉。确认：金叉后延续2日。

均线粘合突破：MA5、MA10、MA20最大差值百分比 < 2%，今日收盘向上突破且放量 > 1.3倍。确认：次日继续收阳。

假突破过滤：任何买入信号触发后，若持仓3日内跌破买入当天最低价，则强制平仓（止损或微亏）。

8.2 卖出信号（与止损体系重叠时按优先级执行）
均线死叉：MA5下穿MA10。

KDJ超买死叉：K > 80 且 D > 70，然后K下穿D。

跌破布林中轨：收盘价 < 布林中轨(20,2)。

MACD顶背离：价格创20日新高，MACD柱未创新高，随后死叉。
