## 无前视偏差保证

1. **信号**: 使用 T close 时刻已知的数据 (5日跌幅、RSI6)
2. **状态检测**: 对 hysteresis 结果滞后 3 天, 避免使用未来数据
3. **T+1 结算**: T 日信号 -> T+1 开盘成交, 持仓首日不卖
4. **涨跌停**: 双校验 (is_limit_up/down + OHLC 推算)
5. **停牌**: volume=0 检测 + 20 天最大挂单
6. **除权除息**: 单日 > 15% 异常用前日 close 公平退出

## 关键工程注意 (避免聚宽运行失效)

详见 D:\claude-quant\JQ\createBase\ATOS_MR_v2_JQ_GUIDE.md

核心 8 个 quirk:
1. **Python 3.6 兼容**: 不能用 list[str], dict[str, X], X | Y, match/case
2. **np.isnan 误判**: JQ 引擎对合法正数返回 True, 用 np.isfinite 代替
3. **cd.get() 永远 None**: get_current_data() 是 lazy loading, 用 cd[s] + try/except
4. **attribute_history(skip_paused=True) 过度过滤**: 改 skip_paused=False + n=70
5. **科创板 (688) 限价单**: 不能用市价单, 必须 LimitOrderStyle(last_price * 1.005)
6. **order_target_value 内部股数 0 假报**: 改 order(stock, delta_shares) 直接传股数
7. **cd[stock] 退市抛 KeyError**: 用 try/except KeyError 守护
8. **high_limit / low_limit 可能为 0**: 加 > 0 守护
